# stringweave

Web app that turns any image into string art. Upload a photo, set how many pins and strokes you want, and watch the drawing build itself — one straight line at a time — until the image emerges from the tangle.

## Features

**Generate** — Upload JPEG/PNG/WebP, choose 120–640 pins and up to 5000 strokes, run, export the nail sequence as `.txt`.

**Player** — Import a `.txt` sequence and scrub or play it back stroke by stroke.

## Stack

Next.js · TypeScript · Tailwind CSS · shadcn/ui · Web Workers

## Run

```bash
npm install
npm run dev
```
